Welcome

Edit Pad is a basic word processor application for windows computers.

Edit Pad is well preferred for taking quick notes and text-based writing.


Special features

1. Spell Check - Spell Check function is available for checking spelling mistakes.

2. Symbols - Let you insert a different kinds of Symbols in the document.

3. New Window - Let you open a new application window.


Supporting OS

Windows Vista / 7 / 8 / 8.1 / 10 / 11


Additional Requirements

Microsoft .NET Framework 4.5 Download

Note : If you need a full-featured word processor, you should use Microsoft Word / LibreOffice Writer etc.
